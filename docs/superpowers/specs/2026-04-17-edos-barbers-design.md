# Edos Barbers — V1 Frontend Design Spec
**Date:** 2026-04-17
**Status:** Implemented

## Overview
React Native / Expo mobile app for Edos Barbers, a premium barbershop in Vienna.
Editorial Luxury design: cream base, charcoal text, gold accents.

## Design System
- **Background:** Cream #F5F0E8
- **Text:** Charcoal #1A1A1A
- **Accent:** Gold #C9A84C
- **Cards:** Warm Grey #E8E2D9 / White #FFFFFF
- **Muted:** #6B6258
- **Fonts:** Playfair Display (display/headings) + Inter (body/labels)

## Screens
1. HomeScreen — Hero, CTAs, news carousel, hours, Instagram strip
2. ServicesScreen — Filter tabs, service cards, sticky book button
3. GalleryScreen — Filter tabs, 2-column placeholder grid
4. AboutScreen — Brand story, team scroll, values strip
5. ContactScreen — Contact info, hours, map placeholder, social
6. MembershipScreen — 3-tier plans (Silber/Gold/Diamant)
7. BookingScreen — Centered dead "Jetzt Buchen" button

## Navigation
Bottom tab (5 tabs) + Stack for Membership and Booking screens.

## Language
German (de-AT). All copy is static for V1.

## Image Strategy
All images are warm grey placeholders with gold border + scissors icon.
Owner will provide real images to replace them.
